¿Como empezar en el desarrollo web en 2020?

Para empezar, el desarrollo web se divide en dos partes: lado del cliente (Front-End) y lado del servidor (Back-End). Acá abajo te dejo un ejemplo muy simplificado de los dos. Tienes que saber que el desarrollo web es muy amplio y este articulo solo es para guiarte por donde empezar.

Front-End

Es la parte visual de la aplicación web, es la parte que ve el usuario. Este conjunto crea la experiencia del usuario y recibe los datos del Back-End para mostrarlo de manera que el usuario lo entienda.

Para ser desarollador Front-End tienes que saber: HTML, CSS y JavaScript con ellos puedes hacer el Front-End de una Web , pero hay tecnologías que deberías aprender como Frameworks y Librerías. Estas tecnologías te harán el trabajo mucho más fácil y te va a ahorrar mucho tiempo.

Back-End

El backend es la parte del desarrollo web que se encarga de que toda la lógica de una página web funcione. Esta parte es la que se encarga de almacenar datos(imagenes, textos, archivos, etc), llevar los datos al Front-End y devolverlos cuando sea requerido.

Estas son alguna cosas que tienes que aprender del Back-End-End:

  • Base de datos y caché
  • APIs(REST)
  • Lenguajes de Servidor

¿Cual debería aprender?¿Desarrollo Back-End o Front-End?

Deberías aprender el que más te guste, pero yo te recomendaría aprender los dos o aprender algún Stack, ya que va a beneficiarte tanto profesionalmente como laboralmente. Acá abajo te dejo lo que deberías aprender de cada uno.

Front-End

Frameworks y Librerías JavaScript: Un framework es una herramienta de desarrollo web que se define como una aplicación o conjunto de módulos que permiten el desarrollo rápido y ágil de aplicaciones ya que tienen librerías y/o funcionalidades ya creadas. Básicamente hace el trabajo de desarollador mas rápido y fácil. Estos son algunos Frameworks JavaScript más populares:Angular, React y Vue. Los tres son muy buenos, te recomiendo que los investigues para sepas cual es el que más te conviene o que más te interesa para lo que quieres hacer.

Frameworks CSS: Un framework de CSS es una biblioteca de estilos genéricos que puede ser usada para implementar diseños web(como botones, iconos, targetas, etc). Los frameworks CSS vienen con clases ya creadas, esto te va ahorrar mucho tiempo al crear una página web. Uno de los Frameworks CSS más Populares es Bootstrap.

Pre Procesadores de CSS: Un preprocesador de CSS es una herramienta que nos permite escribir pseudo-código CSS que luego será convertido a CSS real . Básicamente hace que este lenguaje de diseño tenga variables, condiciones, bucles o funciones como un lenguaje como JavaScript.
Estos son algunos de los mas conocidos Sass, Less y Stylus.

Back-End

Lenguaje de Base de Datos: Además de los lenguajes de programación, tienes que aprender un lenguaje de bases de datos para poder almacenar la información, que es lo mas importante de cualquier aplicación web .

Lenguaje del lado del servidor: Se conoce como lenguaje del lado del servidor a aquel que se ejecuta en el servidor web, inmediatamente antes de que el sitio web se envíe a través de Internet al usuario. Este se encarga de recibir solicitudes de clientes que solicitan páginas HTML u otro tipo de archivos como imágenes y entregarlos al usuario. Algunos lenguajes de servidor sonPHP, Node.JS (programación backend en JavaScript), Python, entre otros.

Frameworks y Librerías: El Framework sigue siendo una grán ayuda al desarrollar como en el Front-End pero en este caso todos los frameworks tienen la principal tarea de crear un servidor web.

Te aconsejo Aprender un Stack

¿Que es un Stack?

Básicamente, es una lista de todos los servicios tecnológicos utilizados para construir y ejecutar una sola aplicación. Facebook, por ejemplo, está compuesto por una combinación de frameworks de codificación y lenguajes, entre los que se incluyen JavaScript, HTML, CSS, PHP y ReactJS. Este es el “stack tecnológico” de Facebook.

Por lo general un Stack tiene todo nescesario para crear una web(o aplicación) y te será más fácil buscar un Stack, a tener que buscar Framework por Framework y lenguaje por lenguaje.

Stacks más populares:
  • MEAN Stack
  • MERN Stack
  • MEVN Stack
  • LAMP Stack

Te recomiendo buscar más información sobre estos u otros stacks.
